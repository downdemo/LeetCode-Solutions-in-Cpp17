* 动态规划

```
$$$
$$b
$a@

到达 @ 之前，必然先到达 a 或 b
因此到达 @ 的路径数 = 到达 a 的路径数 + 到达 b 的路径数
```

* 对于位于边界上的位置，只可能有一条直线路径，因此可作为初始解，再用其计算其他位置

```
@@@    111    111    111
@$$ => 1$$ => 123 => 123
@$$    1$$    1$$    136

结果为6
```

* 解法如下

```cpp
class Solution {
 public:
  int uniquePaths(int m, int n) {
    vector<vector<int>> dp(m, vector<int>(n));
    for (int i = 0; i < n; ++i) {
      dp[0][i] = 1;
    }
    for (int i = 0; i < m; ++i) {
      dp[i][0] = 1;
    }
    for (int i = 1; i < m; ++i) {
      for (int j = 1; j < n; ++j) {
        dp[i][j] = dp[i - 1][j] + dp[i][j - 1];
      }
    }
    return dp[m - 1][n - 1];
  }
};
```
